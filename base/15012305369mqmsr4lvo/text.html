Путь первый

В консоли в корне проекта набрать

composer install name_modul

Компостер сам найдет модуль на гите и скачает его


Путь второй

В файл composer.json добавить зависимость.

Обычно у всех модулей, которые предуматривают установку через комостер в коде в корне лежит файлик composer.json

Код из него копируем в наш файлик(из нашего ничего удалять не надо, просто добавляем)

Затем в консоли в корне проекта набираем

composer install

И компостер загрузит нужные модули(зависимости)
